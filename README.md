## General Information
- You can make many different ToDo lists.
- It helps keep track of things.
- I made it because I was bored.


## Technologies Used
- Python 3.9.9
- tkinter
- sqlite3


## Features
- Create a ToDo list.
- Add items to your list.
- Mark items done/undone with a double click.
- Copy a ToDo list.
- Clear a ToDo list.
